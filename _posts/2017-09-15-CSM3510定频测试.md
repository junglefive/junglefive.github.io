# 3.下位机

下位机默认串口为：`BuadRate = 19200`, `Data = 8`, `Stop = 1`, `Parity = no`.
 ![上位机软件主界面](picture/csm3510/下位机示意图.jpg)

# 4. 操作步骤

 - 如上正确配置好`SerialPort`
 - 点击`Open`,如果正确配置会显示

```
    check serialport
    choosed    #2402
    connecting...
    connected #COMX
```
点击`Send`,成功则显示（`#x`，其中`0 < x <39`，实际上就是代表通道索引值）

```
    send    #0
    recieve #0

```
默认是`TX`模式，选择`RX`可更换模式,以下表示配置成功.

>Rx

```
     send   #RX_FLAG
     set    RX

```

>Tx

```
 send   #TX_FLAG
 set    TX

```
